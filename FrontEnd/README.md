## To Launch Website

Open in Dev Container <br>
cd backend <br>
flask --debug run <br>

in New terminal <br>
cd frontend/client <br>
npm run serve <br>

*** Make sure to have 2 terminals running, one in the BackEnd directory and the other in the FrontEnd/Client/ directory to properly do query***
